# Ontologies Output Directory

This directory contains agent training ontology files generated by the agent training pipeline.

## Purpose

Agent training ontologies are JSON files that contain domain knowledge extracted from Wikidata and Wikipedia for specific QIDs (e.g., Q17167 for Roman Republic).

## File Format

Ontology files are named: `{QID}_ontology.json`

Example: `Q17167_ontology.json`

## Generation

To generate an ontology for a specific QID:

```bash
python scripts/reference/agent_training_pipeline.py Q17167
```

See `START_HERE.txt` in the project root for complete agent training workflow.

## Contents

Each ontology file contains:
- **subject_concept_id**: Canonical identifier
- **wikidata_properties**: Fetched properties from Wikidata
- **wikidata_backlinks**: Related entities
- **wikipedia_toc**: Table of contents structure
- **domain_ontology**: Discovered sub-concepts by facet
- **confidence_baselines**: Evidence-based confidence scores

## Loading to Neo4j

To load a trained ontology into Neo4j:

```python
from scripts.reference.load_trained_ontologies_to_neo4j import TrainedOntologyLoader
import json

loader = TrainedOntologyLoader('bolt://localhost:7687', 'neo4j', 'password')
with open('ontologies/Q17167_ontology.json') as f:
    ontology = json.load(f)
loader.load_ontology_to_neo4j(ontology)
```
