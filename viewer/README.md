# Chrystallum JSX Viewer

Minimal Vite + React setup to preview the Chrystallum JSX architecture and constitution diagrams in a browser.

## Quick start

```bash
cd viewer
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Components

| Tab | File | Description |
|-----|------|-------------|
| Discipline Universe | `Key Files/chrystallum_discipline_universe.jsx` | 18 facets, disciplines, repository index |
| TOC Constitution | `Key Files/chrystallum_toc_constitution.jsx` | Transition of Care agent constitution |
| Architecture | `Key Files/chrystallum_architecture.jsx` | System architecture diagram |

## Scripts

- `npm run dev` — Start dev server (hot reload)
- `npm run build` — Production build to `dist/`
- `npm run preview` — Preview production build locally
